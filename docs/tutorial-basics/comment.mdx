## lấy repository API key

Đăng nhập vào [GraphQL API Explorer](https://docs.github.com/en/graphql/overview/explorer). 
và dán đoạn code bên dưới , nhớ thay nameOfYourGitHubAccount , nameOfCreatedRepository tương ứng 
**lưu lại 2 thông tin id của repository và General tương ứng**  

   ```
query { 
  repository(owner: "nameOfYourGitHubAccount", name:"nameOfCreatedRepository"){
    id
    discussionCategories(first:10) {
      edges {
        node {
          id
          name
        }
      }
    }
  }
}
```
